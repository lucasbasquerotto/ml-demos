# Snake Game

A classic Snake game built with HTML5, CSS3, and JavaScript that runs entirely in the browser.

## Features

- **Smooth gameplay** with increasing difficulty
- **Score tracking** with persistent high score (saved in localStorage)
- **Responsive design** that works on different screen sizes
- **Keyboard controls**: Arrow keys or WASD
- **Game controls**: Start, Pause/Resume, Reset
- **Visual feedback** with animated snake head (eyes that follow direction)
- **Clean UI** with modern styling

## How to Play

1. Open `index.html` in any modern web browser
2. Click "Start Game" or press Enter/Space to begin
3. Use arrow keys (‚Üë ‚Üì ‚Üê ‚Üí) or WASD to control the snake
4. Eat the red food to grow and score points
5. Avoid hitting the walls or yourself
6. Press Space to pause/resume during gameplay

## Game Mechanics

- Each food eaten: +10 points
- Snake grows by 1 segment per food
- Game speed increases every 50 points
- High score is saved locally in your browser

## Technologies Used

- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS3 for styling and animations
- LocalStorage for high score persistence

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `game.js` - Game logic and controls
